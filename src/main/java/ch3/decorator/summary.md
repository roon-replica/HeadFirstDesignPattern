### 정의
- 객체에 추가 요소를 `동적으로`(컴파일 타임에 말고라고 보면 될 듯) 추가 가능   
  상속을 이용한 서브클래스를 보다 더 유연하게 기능을 확장 가능하다!
- 생각해보면 taskExecutor를 try-finally로 감싼것도 데코레이터 패턴이겠네.  
  그 메서드 이름도 taskDecorator였고..!!!
